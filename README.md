# gw2armory.com ![Build Status](https://travis-ci.org/madou/gw2armory.com.svg)

> Mirror mirror on the wall.. who is the prettiest of them all.

## Usage

### Install dependencies
```
npm install
```

### Run dev build
```
npm start
```

### Run prod build
```
npm run start:prod
```

### Run tests
```
npm test
npm run test:auto
```

### Build dev
```
npm run build:dev
```

### Build prod
```
npm run build:prod
```
