If you feel like contributing give me a message and we can talk.

## Usage

### Install dependencies
```
npm install
```

### Run dev build
```
npm start
```

### Run prod build
```
npm run start:prod
```

### Run tests
```
npm test
npm run test:auto
```

### Build dev
```
npm run build:dev
```

### Build prod
```
npm run build:prod
```
