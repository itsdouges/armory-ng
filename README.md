> Bringing you `that shit you do like`, one commit at a time.

## Usage

If you're playing on windows, add `:win` after each step, for example: `npm run start:win`.

### Install dependencies
```
npm install
```

### Run dev build
```
npm run start
```

### Run prod build
```
npm run start:prod
```

### Run tests
```
npm run test
npm run test:auto
```

### Build dev
```
npm run build:dev
```

### Build prod
```
npm run build:prod
```
